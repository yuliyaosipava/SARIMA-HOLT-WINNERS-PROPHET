# SARIMA-HOLT-WINNERS-PROPHET
# Анализ временного ряда и прогнозирование с помощью SARIMA, Holt-Winters и Prophet

Этот проект представляет собой комплексный анализ месячного временного ряда и построение прогнозных моделей. Основная цель — выявление аномалий, выбор оптимальной модели и построение прогноза с использованием методов эконометрики и машинного обучения.

## 📁 Структура проекта

Проект состоит из Jupyter Notebook (`Final.ipython.ipynb`), который включает в себя следующие этапы:

1.  **Загрузка и подготовка данных**: Загрузка данных из Excel-файла (`Gomel1000.xlsx`), преобразование индекса промышленного производства во временной ряд с месячной частотой.
2.  **Обнаружение выбросов**: Использование нескольких статистических методов для идентификации аномалий во временном ряду (Z-score, IQR, Hampel filter, STL residuals, SARIMA residuals).
3.  **Обработка выбросов**: Применение методов сглаживания и интерполяции для устранения влияния аномалий на модели.
4.  **Сравнение моделей**:
    - **SARIMA (Seasonal Autoregressive Integrated Moving Average)**: Классическая эконометрическая модель.
    - **Holt-Winters**: Модель экспоненциального сглаживания с трендом и сезонностью.
    - **Prophet**: Модель от Facebook, разработанная специально для прогнозирования временных рядов с сезонностью.
5.  **Валидация и тестирование**:
    - Разделение данных на обучающую и тестовую выборки (hold-out валидация).
    - Скользящая перекрёстная проверка (rolling expanding window cross-validation).
    - Сравнение моделей по метрикам (MAE, RMSE, MAPE).
6.  **Диагностика остатков**: Проверка остатков моделей на нормальность, автокорреляцию, гетероскедастичность и ARCH-эффекты.
7.  **Прогнозирование**: Построение прогноза на 12 месяцев с помощью выбранной модели (SARIMA).
8.  **Визуализация**: Построение графиков остатков, ACF/PACF, QQ-plot, гистограмм и сравнение фактических и предсказанных значений.

## 📊 Используемые модели и методы

### Эконометрические модели
- **SARIMA(p,d,q)(P,D,Q,s)**: Использовалась для моделирования и прогнозирования. Параметры:
  - `order=(1,1,1)`
  - `seasonal_order=(1,0,1,12)`
- **Holt-Winters**: Аддитивная модель тренда и сезонности.

### Модель машинного обучения
- **Prophet**: С годовой сезонностью, без недельной и дневной.

### Методы детекции и обработки выбросов
- **Z-score** (на логарифмированных данных)
- **IQR (Interquartile Range)** (на логарифмированных данных)
- **Hampel filter** (скользящее окно с медианой)
- **STL Residuals** (остатки сезонно-трендовой декомпозиции)
- **SARIMA Residuals** (остатки предварительной модели SARIMA)

### Валидация
- **Hold-out**: Последние 12 месяцев (2025-2026) использовались как тестовая выборка.
- **Rolling Expanding Window**: Для оценки стабильности модели во времени.

### Метрики
- **MAE (Mean Absolute Error)**
- **RMSE (Root Mean Squared Error)**
- **MAPE (Mean Absolute Percentage Error)**

## 📈 Основные результаты

В результате анализа было проведено сравнение трёх моделей. Ключевые выводы включают:

- Наиболее стабильные и качественные остатки (близкие к нормальному распределению, отсутствие автокорреляции) показала модель **Holt-Winters**.
- Модели **Prophet** и **SARIMA** также показали хорошие результаты, но имели некоторые проблемы с остатками (автокорреляция, нестационарность).
- Наибольшую точность прогноза (`MAE=36.15`, `RMSE=39.29`, `MAPE=0.39%`) показала модель на **сглаженных данных (smooth)**.
- Модель **Prophet** показала достойные результаты (`MAE=281.56`, `RMSE=347.59`, `MAPE=2.19%`).
